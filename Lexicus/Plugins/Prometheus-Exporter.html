What Is Prometheus Exporter?

Exports server statistics like CPU and RAM usage, chat, logins, chunks, entities, uptime, and many others for use in Prometheus.

How Do We Use Prometheus Exporter?

We link Prometheus with a self hosted service called Grafana which displays this data in graphs and charts. You can access our stats page here!
