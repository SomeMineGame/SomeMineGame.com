What Is DiscordSRV?

DiscordSRV links a Discord channel to Minecraft's chat to allow messaging between the game and members on Discord. It also allows the linking of a Discord channel to the console to allow moderation and remote commands.

How Do We Use DiscordSRV?

We use it to chat with players while away from the game, and to moderate the server when unable to access the server.
