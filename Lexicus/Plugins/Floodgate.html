What Is Floodgate?

Allows Bedrock players to join Geyser supported servers with their Bedrock account instead of needing to use a Java account. This keeps Java and Bedrock fully seperate, and with how Minecraft Java and Bedrock are now bundled, gives all players the option to use an alt account.

How Do We Use Floodgate?

We use it to fully keep Java and Bedrock users separate. It allows allows for Bedrock users to continue using their Marketplace skins with other Bedrock users.
