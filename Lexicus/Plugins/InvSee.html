What Is InvSee++?

Allows for moderation of online and offline players' inventories. Admins can give, take, or clear items for players if needed.

How Do We Use InvSee++?

We only use this plugin if it's needed due to a bad actor trying to use exploits to either crash the server or chunk ban players, although we'd use it after banning them first.
