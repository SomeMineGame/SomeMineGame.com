What Is DiscordSRV?

DiscordSRV connects Discord and Minecraft together. It allows for the Minecraft chat to post live into a Discord channel, and for that same channel to send Discord messages to Minecraft. These will include the usernames of who sent the message. It also allows for the server console to be linked to a channel for admins to remotely view.

How Do We Use DiscordSRV?

We use DiscordSRV to allow for Minecraft and Discord communication so offline members can still talk to online players, and allow for easier messaging for those in game (especially mobile and console users).
