What Is ProtocolLib?

ProtocolLib is a library that some plugins use for easier management and creation. It has some pre-made functions and features that speed up plugin creation and ensures compatibility.

How Do We Use ProtocolLib?

We use ProtocolLib to make sure plugins run correctly and to their fullest capabilities. Some plugins only have limited features without it, so this ensures we can use all features and prevent bugs from occuring.
