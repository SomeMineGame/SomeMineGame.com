What Is Dynmap?

Dynmap is a Google Maps style plugin that allows for configurable views of your server. This could be a top down view, or an angled view of the world.

How Do We Use Dynmap?

We use Dynmap to show the current state of the server. This allows for the viewing of builds before a recap video is released, or if it isn't included in a video.
