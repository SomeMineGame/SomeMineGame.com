What Is GeyserMC?

GeyserMC is a proxy that translates data between Java servers and Bedrock clients. This allows Bedrock devices to log into a Java account and join a Java server. There isn't any lag or delay caused by this for anyone including Bedrock users, so the experience is seamless. We use Floodgate to bypass the Java account requirement, allowing Bedrock users to use their current account.

How Do We Use GeyserMC?

We use GeyserMC to expand our player base to those outside of Java. This allows for those who prefer Bedrock or don't own a computer to join our Java server.
