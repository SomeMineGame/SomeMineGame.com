What Is Floodgate?

Floodgate allows Bedrock users joining Java servers with its parent plugin, GeyserMC, to join with their Bedrock account instead of logging in with a Java account. This is useful if there are Bedrock users who don't own Java or if you want players to have different accounts (such as an alt) based on the other edition.

How Do We Use Floodgate?

We use Floodgate to keep the editions separate, and allow players that own only Bedrock to still have access to the server.
