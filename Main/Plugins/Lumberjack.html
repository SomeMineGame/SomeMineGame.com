What Is Lumberjack?

Lumberjack is a semi-realism plugin that adds tree gravity on a per user basis. It also offers faster leaf decay.

How Do We Use Lumberjack?

Wee use Lumberjack to help tear down trees and add a bit of a realistic feature to the server.
