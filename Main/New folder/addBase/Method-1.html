Command

&addBase xcoord ycoord zcoord dimension

Usage

Add your base to the system for future reference by yourself or other players using the X, Y, Z, and Dimension you're in.

Permission

Everyone

Example

&addBase 100 67 -250 Nether

This will add the base at the in-game coordinates of 100 67 -250 and put the icon in the Nether on Dynmap.

Enabled

True
