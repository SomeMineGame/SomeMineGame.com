Command

&addBase xcoord ycoord zcoord dimension

Usage

Adds your base to Discord and the live map with the dimension provided. Dimensions can be Overworld, Nether, or End.

Permission

Everyone

Example

addBase 500 70 250 Nether

This would add the player's base to those coordinates and put a house icon there on the map in that dimension.

Enabled

Yes
