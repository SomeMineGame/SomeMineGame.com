Command

&addDonator user tier months

Usage

Adds a user to the donation system for the specified amount of months on a tier from 1-3.

Permission

Bot Admin

Example

&addDonator SomeMineGame 2 3

This will add SomeMineGame to the Donators $5 role and benefits for 3 months.

Enabled

No
