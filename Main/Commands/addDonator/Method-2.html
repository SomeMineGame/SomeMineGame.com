Command

&addDonator user 4 months amount

Usage

Adds a user to the donation system on tier 4 (custom) for the specified amount of time. Amount must be higher than 10.

Permission

Bot Admin

Example

&addDonator SomeMineGame 4 2 30

SomeMineGame paid $30 for 2 months ($15/month), so he'll get the custom donator's role along with tier 3's benefits.
