Command

&amendPoll title message

Usage

Creates a vote for adding an amendment to the Constitution (Rules) with the option to accept it, change it, or deny it.

Permission

Bot Admin

Example

&amendPoll "No Inflation Sell Raise" "Stops players from raising the inflation rate to get an increased yield when selling items, then lower it again"

This will create a poll for players to choose if they want to accept, change, or deny this new ban on an exploit to gain more money by raising the inflation rate before selling.

Enabled

Yes
