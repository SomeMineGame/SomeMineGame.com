Command

&cmd command

Usage

Send a command to Minecraft remotely and directly from Discord.

Permission

Bot Admin

Example

&cmd tp SomeMineGame 0 70 0

This will send a tp command to the Minecraft server, and if SomeMineGame is online will teleport him to the coordinates of 0 70 0.

Enabled

Yes
