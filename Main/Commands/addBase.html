Command

addBase

Description

Adds your base so others and yourself can see its coordinates easier. Integrates with Dynmap. Dimension is by default Overworld.

Permission

Everyone

Syntax

&addBase <xcoord> <ycoord> <zcoord> [dimension]

Possible Combinations (Click Row For More Information)

&addBase xcoord ycoord zcoord
&addBase xcoord ycoord zcoord dimension
