Command

buy

Description

Buy an available item from the shop using money in your wallet and have it given to you directly in Minecraft. Must be in Minecraft to run.

Permission

Everyone

Syntax

&buy [seller] [amount] <item>

Possible Combinations (Click Row For More Information)

&buy item
&buy amount item
&buy seller item
&buy seller amount item
