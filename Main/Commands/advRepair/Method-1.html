Command

&advRepair player file

Usage

Recreates a specified file if a user is getting errors or it wasn't properly updated.

Permission

Bot Admin

Example

&advRepair SomeMineGame economy

This will take the current data out of SomeMineGame's economy file, delete it, create a new one with default values, and put the data back into it. Missing data will now be the default values.

Enabled

Yes
