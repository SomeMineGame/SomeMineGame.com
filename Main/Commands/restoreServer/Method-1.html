Command

&restoreServer username saveName

Usage

After using the username as verification and confirmation, his will move the current server's files to a new folder named after the current date, then move the old server under saveName back. Note: This is a dangerous command as old files might be out of date and will therefore be broken.

Permission

Server Owner

Example

&restoreServer SomeMineGame "CreativeServer"

This will use SomeMineGame's name as verification and confirmation before moving all the files to a folder named the current date (for example: "June 14, 2023"). After that is done, it will pull back "CreativeServer"

Enabled

Yes
