Command

&resetServer username resetName

Usage

Resets all server files and moves them to a new directory under resetName. This is useful if you want to reset the Discord server and Minecraft server for a fresh start.

Permission

Server Owner

Example

&resetServer SomeMineGame The Best Server

This will take the username SomeMineGame as verification and confirmation, then if it matches will move all the files to a folder called "The Best Server" if it isn't already there. It will finally remake every file with the default values.

Enabled

Yes
