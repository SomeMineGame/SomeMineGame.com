Command

&charity Info Name

Usage

Get the information related to a charity. If you're the creator, you'll be dm'd extra stats.

Permission

Everyone

Example

&charity Info "Project Funding"

If creator, this user will get a message with the name purpose, organizer, goal, raised, donations, distributed, funds, active status, and the activatable status. If not, a general message will be sent with the name, purpose, organizer, goal, raised, donations, distributed, and the active status.

Enabled

Yes
