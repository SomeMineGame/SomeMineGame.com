Command

&charity Create Name Goal Purpose

Usage

Creates a new charity with the specified name, goal, and purpose. Name is the name of the charity. Goal is the amount you want to raise in the end. The purpose (description) is why you created the charity.

Permission

Everyone

Example

&charity Create "Project Funding" 250000 Help those get money for public structures.

This will create a charity with the name "Project Funding", a goal of $250,000, and a purpose of "Help those get money for public structures".

Enabled

Yes
