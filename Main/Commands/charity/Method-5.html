Command

&charity Give Name Amount "Charity"

Usage

Give a charity the specified amount of money from the specified charity, if you're the creator. (Charity must be in double quotes)

Permission

Everyone

Example

&charity Give "Project Funding" 1500 "NewComers"

This will give NewComers $1,500 from Project Funding's funds.

Enabled

Yes
