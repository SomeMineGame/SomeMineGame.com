Setting Up Simple Voicechat

Firstly, you'll want to go to Curseforge.com and select the orange "Get CurseForge App" button.

Click the "Download Standalone" button for a faster install and less applications needed. If you'd prefer to use Overwolf, you can hit that button instead.

Now click the installation prompt or the file in your downloads folder and follow the installation steps provided. If CurseForge doesn't automatically open, obviously open it yourself.

Select Minecraft in the games list and search "Simple Voice Chat in the search bar. Make sure to select the one with the microphone icon and made by the author henkelmax.

Click install and either make a new profile or select an existing one. After the installation is complete, it will automatically open the profile page. From there you just have to hit play and you'll launch Minecraft like normal.

Once you join a server, you can hit the "V" key to open the option menu. Here, you can change all the settings like volume, noise cancellation, push to talk, and others. There's even an option to record all voice activity of yourself and all nearby players!
