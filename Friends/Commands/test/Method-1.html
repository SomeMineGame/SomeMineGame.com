Command

&test

Usage

Tests if the bot is responsive. If there is a reply, it is working properly. If not, it isn't working properly.

Permission

Everyone

Example

&test

If the bot hit a fatal error and stopped responding, this command could be run. There was no response so it froze and went offline.

Enabled

Yes
