Command

&poll title message option1 option2 option3 option4 option5

Usage

Creates a poll with the number of options provided up to 10, then will add the same amount of corresponding reactions.

Permission

Bot Admin

Example

&poll "End Fight" "When are you guys most able to do the End fight?" Tuesday Wednesday Thursday Friday Saturday

Note, if your options are more than 1 word, they will need to be in quotes. This will create a poll titled "End Fight" and the description "When are you guys most able to do the End fight?", followed by 5 options - Tuesday through Saturday, and 5 reactions for people to vote with.

Enabled

Yes
