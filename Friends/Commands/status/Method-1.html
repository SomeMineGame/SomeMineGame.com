Command

&status

Usage

Give the local status (If the Server is reachable from the bot's network) and global status (If the server is reachable from outside the bot's network).

Permission

Everyone

Example

&status

Say the Minecraft server is online, but there was an issue connecting it to the IP. The bot would return Local: True and Global: False.

Enabled

Yes
